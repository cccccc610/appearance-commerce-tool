# 外观商业化思考小工具 v2.1

> 面向美术策划运营复合型岗位的外观商业化思考工具

---

## 📋 工具概述

本工具是一个综合性的游戏外观商业化思考辅助工具，专门为美术策划运营复合型岗位设计：

- 🎮 **品类自定义**：预设 MOBA/FPS/模拟经营 + 支持自定义输入
- 🎨 **外观类型定制**：根据品类匹配外观类型 + 支持自定义输入
- 📱 **平台榜单竞品**：基于 iOS/安卓/PC 畅销榜 TOP10 选择参考竞品
- 🔍 **针对性内容**：根据外观类型推送专属内容规划、搜索工具
- ⚠️ **针对性检查**：根据外观类型匹配专属风险检查清单

---

## 🆕 v2.1 更新内容

1. **品类选择支持自定义**：除了预设的 MOBA/FPS/模拟经营，支持输入任意游戏品类
2. **竞品选择基于平台榜单**：集成 iOS 畅销榜、安卓畅销榜、PC 热门榜 TOP10 游戏
3. **外观类型支持自定义**：根据品类自动匹配外观类型，也支持自定义输入
4. **针对性内容推送**：根据选择的外观类型，推送对应的内容规划维度和搜索工具
5. **针对性风险检查**：不同外观类型对应不同的风险检查清单
6. **简化流程**：取消生成报告环节，专注于分析和检查


## 🔧 功能模块

### 1. 决策树引导

通过问答式引导，帮助你做出关键决策：

| 决策树 | 功能 | 适用场景 |
|--------|------|----------|
| **英雄/角色选择** | 确定皮肤的目标英雄 | 选择要制作皮肤的角色 |
| **品质等级定位** | 确定皮肤的品质等级 | 规划皮肤制作规格 |
| **定价决策** | 计算合理的定价区间 | 制定价格策略 |
| **主题选择** | 确定皮肤的主题方向 | 规划皮肤风格 |

### 2. 检查清单验证

三大维度的完整性检查：

| 检查类型 | 检查内容 |
|----------|----------|
| **完整性检查** | 设计定位、美术内容、定价策略、制作排期、运营规划 |
| **合理性检查** | 定价合理性、排期合理性、目标合理性 |
| **风险检查** | 设计风险、技术风险、运营风险、合规风险 |

### 3. 竞品参考

提供同类游戏的商业化方案参考：

| 游戏品类 | 竞品案例 |
|----------|----------|
| **MOBA** | 王者荣耀、英雄联盟、DOTA2、决战平安京 |
| **FPS** | CS2、VALORANT、Apex英雄、穿越火线 |
| **模拟经营** | 动物森友会、原神家园、光·遇、梦幻西游 |

### 4. 美术内容参考

根据品类梳理美术内容需求：

- **主题题材**：节日主题、IP联动、原创系列、文化主题、风格主题
- **动画需求**：待机动画、回城动画、技能动画、胜利动画、表情动作
- **特效需求**：技能特效、回城特效、击杀特效、死亡特效、被动特效
- **故事背景**：皮肤名称、背景故事、角色设定、系列关联
- **音效需求**：语音包、技能音效、回城音效、击杀音效、主题BGM

## 💡 使用建议

### 使用流程

```
1. 选择游戏品类和外观类型
2. 运行决策树引导（或选择一键运行）
3. 运行检查清单验证
4. 查看竞品参考和美术内容
5. 生成方案报告
6. 根据报告完善方案
```

### 最佳实践

- **首次使用**：建议使用"一键运行"体验完整流程
- **迭代优化**：根据检查清单结果，逐项完善方案
- **风险规避**：重点关注"严重风险"和"高风险"预警项
- **竞品对标**：参考成功案例的关键成功因素
